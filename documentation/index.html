Smart School Management Template — Complete Setup Guide

v1.0.0 Last Updated: 2025

Getting Started

EduTrack is a complete school management dashboard template. Follow these steps to get started:

  • 1. Extract the ZIP file to your project folder
  • 2. Open index.html in your browser
  • 3. Customize colors, text, and images
  • 4. Deploy to your hosting provider

File Structure

  • index.html — Main landing page
  • pages/login.html — Login page
  • assets/css/style.css — Main styles
  • assets/css/dark-mode.css — Dark theme
  • assets/css/responsive.css — Mobile responsive
  • assets/js/main.js — Core functionality
  • assets/js/charts.js — Charts with Chart.js
  • assets/js/theme-switcher.js — Dark/light toggle

Customization

Colors: Edit :root variables in style.css

  • --primary: #4f46e5; — Main brand color
  • --success: #22c55e; — Success color
  • --danger: #ef4444; — Danger color

Logo: Replace the icon in .nav-brand and .auth-logo

Responsive Breakpoints

  • Desktop: > 1024px — Full layout
  • Tablet: 768px - 1024px — 2-column grid
  • Mobile: 480px - 768px — Single column
  • Small Mobile: < 480px — Optimized spacing

Charts

The template uses Chart.js for interactive charts:

  • Hero mini chart — heroChart
  • Main bar chart — mainChart
  • Performance line chart — performanceChart

To update data, modify the data arrays in charts.js

Dark Mode

Dark mode is automatically enabled based on system preference. Users can toggle manually:

  • Click the 🌙/☀️ button in the top-right corner
  • Preference is saved in localStorage
  • Dark mode styles in dark-mode.css
