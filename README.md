# 📚 EduTrack - Smart School Management Template

![EduTrack Preview](https://via.placeholder.com/1200x630/4f46e5/ffffff?text=EduTrack+Preview)

## 🎯 Overview

EduTrack is a premium, fully responsive school management dashboard template. Perfect for schools, colleges, and educational institutions to manage students, subjects, results, and report cards.

## ✨ Features

- ✅ **100% Responsive** — Works on all devices
- ✅ **Dark/Light Mode** — Toggle with one click
- ✅ **Interactive Charts** — Beautiful data visualization
- ✅ **Login Page** — Ready-to-use authentication
- ✅ **Clean Code** — Well-organized and commented
- ✅ **Cross-Browser** — Chrome, Firefox, Safari, Edge
- ✅ **Font Awesome** — 2,000+ icons included
- ✅ **Google Fonts** — Inter font family
- ✅ **No Dependencies** — Pure HTML/CSS/JS
- ✅ **Documentation** — Complete setup guide

## 🚀 Quick Start

1. Download the template
2. Open `index.html` in your browser
3. Customize colors in `assets/css/style.css`
4. Deploy to your hosting provider
